# Net Practice
This project was created as part of the [42](https://www.42sp.org.br/) curriculum by `mhidani`.

## Description 📃

*net_practice* is a project focused on learning and practicing fundamental 
concepts of computer networking. Its goal is to develop an understanding of how 
devices communicate on a network, covering topics such as IP addressing, 
subnets, routing, and network configuration.

Throughout the course, practical exercises are provided that simulate 
real-world network configuration scenarios, encouraging problem-solving and 
logical thinking. Users must analyze different topologies and correctly 
configure network parameters to ensure communication between devices.

| These exercises can be found in the compressed file provided by Intra.

This course is ideal for anyone who wants to strengthen their knowledge of 
networking, especially when it comes to essential concepts such as:

- IP Addresses (IPv4)
- Subnet masks
- Gateways
- Routing tables
- Network segmentation

## Instructions 📗

There are files in `JSON` format containing the 10 proposed exercises, which 
can be viewed in the files listed in the repository.
